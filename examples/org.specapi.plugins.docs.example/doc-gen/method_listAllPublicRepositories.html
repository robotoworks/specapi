List All Public Repositories

GET https://api.github.com/repositories

This provides a dump of every public repository, in the order that they were created. Note: Pagination is powered exclusively by the since parameter. Use the Link header to get the URL for the next page of repositories.

Parameter Comments
since The integer ID of the last Repository that you've seen.

200 OK
